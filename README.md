# Customers List (customers_challenge-frontend)

Listagem de clientes da Juntos Somos +


## Rodando o projeto

### Instale as dependencias
```bash
yarn
```

###Execute
```bash
yarn start
```

### Para rodar os testes
```bash
yarn test
```

O projeto irá rodar por padrão em ```localhost:3000```




### Tecnologias:
- Quasar Framework
- Yarn
- Scss
- Material Icons
- Jest
- Node
